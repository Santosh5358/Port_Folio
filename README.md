## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No build process required.

## Customization

### Update Personal Information

1. Edit `index.html` to update:
   - Name and title
   - Contact information (email, LinkedIn, GitHub)
   - Project descriptions and outcomes
   - Skills and experience
   - Work experience details

### Update GitHub Projects

All project links point to: `https://github.com/Santosh5358`

To add specific project links:
1. Find the project card in `index.html`
2. Update the GitHub link in the `project-links` section
3. Add live demo links if available

### Styling

- Modify `styles.css` to change colors, fonts, and layout
- Color scheme is defined in CSS variables at the top of `styles.css`
- Primary color: Java Orange (#f89820)
- Secondary color: Java Blue (#007396)

### Content

- All content is in `index.html`
- Update project cards, skills, and sections as needed
- Add or remove projects as required
